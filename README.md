## Stock Tracker Console App

Simple laravel application which tracks prices and availability of products through the different retailers api.

Released:

- Product availability check (BestBuy Api)
- User notifications
- Save history of tracks
- Module and integration tests


To run application, migrate a database and run: php artisan track.
